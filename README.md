# cn-cnweb-archive

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容来自网络公开信息，以静态文件的形式保存，便于长期访问和参考。

## 目录说明

```
.
├── pages/          # 存放所有归档的 HTML 页面
├── assets/         # 页面引用的静态资源（图片、样式、脚本等）
└── index.html      # 归档页面索引（可选）
```

- `pages/`：每个子目录或文件对应一个独立的归档页面，命名方式通常为 `页面主题/index.html`。
- `assets/`：包含页面所需的公共或私有资源文件。
- `index.html`：可选的导航页面，列出所有归档条目。

## 页面归档说明

- 归档内容为原始页面的完整或精简 HTML 版本，保留主要结构和可读性。
- 部分页面可能经过格式调整，以适应静态托管环境。
- 页面中的外部链接、嵌入资源可能无法正常加载，建议通过本地访问或离线查看。
- 归档不保证与原始页面完全一致，也不代表对内容的认可或推广。

## 维护说明

- 本仓库采用手动或半自动方式更新，无固定发布周期。
- 欢迎通过 Issue 提交归档建议或反馈问题。
- 不接受与内容修改、删除相关的请求，除非涉及法律或安全原因。
- 仓库维护者保留对归档内容的筛选和调整权利。

## 使用方式

你可以直接通过 GitHub Pages 或其他静态托管服务浏览这些页面。本地使用时，克隆仓库后打开 `index.html` 或直接访问 `pages/` 下的文件。

```bash
git clone https://github.com/your-username/cn-cnweb-archive.git
cd cn-cnweb-archive
# 使用浏览器打开 pages/ 下的任意 HTML 文件
```
